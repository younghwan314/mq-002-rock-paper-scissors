# 가위바위보 게임

Java로 만든 콘솔 기반 가위바위보 게임입니다.

## 주요 기능

- 컴퓨터의 가위/바위/보 무작위 선택
- 사용자 선택 입력
- 승리/패배/무승부 판정
- 목표 승수 기반 선승제
- 현재 점수 및 최종 결과 출력
- 게임 재시작

## 게임 규칙

- 사용자와 컴퓨터가 가위/바위/보 중 하나를 선택합니다.
- 승리한 쪽의 점수가 1점 증가합니다.
- 무승부일 때는 점수가 증가하지 않습니다.
- 먼저 목표 승수에 도달한 쪽이 최종 승리합니다.
- 경기가 끝나면 게임을 다시 시작하거나 종료할 수 있습니다.

## 사용 기술

- Java
- `Random`
- `Scanner`
- 배열
- 조건문
- 반복문
- Git과 GitHub

## 배운 점

- 배열의 인덱스를 이용해 무작위 항목을 선택하는 방법
- `String`의 내용을 `equals()`로 비교하는 이유
- 여러 승리 조건을 `&&`와 `||`로 조합하는 방법
- 점수처럼 유지해야 하는 상태를 관리하는 방법
- 안쪽 반복문과 바깥 반복문의 역할을 구분하는 방법
- 공통 코드를 조건문 밖으로 옮겨 중복을 줄이는 방법
- 배열의 길이를 이용해 코드의 확장성을 높이는 방법

## 향후 개선 사항

- 잘못된 사용자 입력 검증
- 목표 승수 선택 기능
- 메서드 분리
- 객체지향 구조로 리팩터링
